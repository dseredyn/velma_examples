Download:

https://github.com/RCPRG-ros-pkg/RCPRG_rosinstall/blob/master/velma_sim_kinematics_setup.sh

Change rights:

chmod 777 velma_sim_kinematics_setup.sh

Source ROS:

source /opt/ros/indigo/setup.bash

Run:

./velma_sim_kinematics_setup.sh ws_velma

In a new terminal:

cd ws_velma/underlay

source devel/setup.bash

roscore

In a new terminal:

cd ws_velma/underlay

source devel/setup.bash

rosrun rviz rviz

In a new terminal:

cd ws_velma/underlay

source devel/setup.bash

roslaunch velma_common velma_fake.launch

In rviz click "Add", choose "RobotModel", click "Ok"

In a new terminal:

cd ws_velma/underlay

source devel/setup.bash

rosrun velma_examples example_control.py

The example script is located in ws_velma/underlay/src/velma_examples/scripts/example_control.py
